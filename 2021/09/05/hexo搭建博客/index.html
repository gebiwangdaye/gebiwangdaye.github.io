GitHub+Hexo 快速搭建个人博客

发表于 2021-09-05 |

Hexo是什么

Hexo 是一个快速、简洁且高效的博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。详情可以访问 Hexo 中文文档

Hexo搭建

搭建步骤

  • GitHub 创建个人仓库
  • 安装 Git
  • 安装 Node.js
  • 安装 Hexo
  • 新建文章

GitHub创建个人仓库

登录到GitHub ，点击GitHub 中的New repository 创建新仓库，仓库名应该为：用户名.github.io 这个用户名是github 的用户名，这是固定写法。

安装Git

可以进入 git 官方网站进行下载。

下载安装成功之后，使用 git bash 设置 user.name 和 user.email 配置信息：

然后找到生成的 .ssh 的文件夹中的id_rsa.pub密钥，将内容全部复制。

打开 github，点击头像选择 Settings ，最后点击 SSH and GPG Keys ，点击 New SSH key 把 id_rsa.pub 密钥的内容复制到里面。

安装 Node.js

可以进入 node 官网进行下载。

注意安装 Node.js 会包含环境变量及 npm 的安装，安装后，在cmd命令行中输入node -v可以检测 node 是否安装成功；在命令行中输入npm -v 可以检测 npm 是 否安装成功。

站点配置

  • 安装 hexo-deployer-git。

  • 修改配置
    参数 描述 默认
    repo 库（Repository）地址
    branch 分支名称 gh-pages (GitHub) coding-pages (Coding.net) master (others)
    message 自定义提交信息 Site updated: {{ now('YYYY-MM-DD HH:mm:ss') }})
    token Optional token value to authenticate with the repo. Prefix with $ to read token from environment variable

  • 生成站点文件并推送至远程库。执行 npx hexo clean && npx hexo deploy

新建文章

在根目录输入命令 npx hexo new [layout] <文章名>

layout 默认情况下有三个模板， 默认情况下是 post

参数 功能 路径
post 新建文章 /source/_posts/
draft 新建草稿 /source/_drafts/
page 新建页面（标签页，分类页等） /source/

输入命令后会在相应的文件夹创建一个 md 文件。

依次输入下列命令将文件发布。

发布到 github 后访问 用户名.github.io 即可访问搭建的博客。

Tips: github.io 现在一般被 DNS 解析为 127.0.0.1 可以修改本地 hosts 文件将全域名解析为固定ip。
